# キー操作
w: カーソールを上方向へ移動

s: カーソールを下方向へ移動

a: カーソールを左方向へ移動

d: カーソールを右方向へ移動

p: セル ON/OFF 切り替え 

f: セル 全てOFF     

g: ゲーム開始    

e: ゲーム開始中断    

o: 閉じる
